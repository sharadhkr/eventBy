## рџљЂ Installation

### Prerequisites
Ensure you have the following installed:
- **Node.js** v16+ ([Download](https://nodejs.org/))
- **npm** v8+ or **yarn**
- **MongoDB** (Local or Atlas)
- **Git**

### Step 1: Clone the Repository
```bash
git clone https://github.com/sharadhkr/eventBy.git
cd eventBy
```

### Step 2: Backend Setup
```bash
cd server
npm install
```

### Step 3: Frontend Setup
```bash
# For user app
cd ../client/user
npm install

# For organizer app
cd ../organizer
npm install

# For admin app
cd ../admin
npm install
```

## вљ™пёЏ Configuration

### Backend Configuration

Create a `.env` file in the `server/` directory:

```env
# Server Configuration
PORT=5000
NODE_ENV=development

# Database
MONGO_URI=mongodb+srv://username:password@cluster.mongodb.net/eventBy

# Firebase
FIREBASE_PROJECT_ID=your-project-id
FIREBASE_PRIVATE_KEY=your-private-key
FIREBASE_CLIENT_EMAIL=your-client-email

# JWT
JWT_SECRET=your-jwt-secret-key
JWT_EXPIRE=7d

# Cloudinary
CLOUDINARY_NAME=your-cloud-name
CLOUDINARY_API_KEY=your-api-key
CLOUDINARY_API_SECRET=your-api-secret

# Razorpay
RAZORPAY_KEY_ID=your-razorpay-key
RAZORPAY_KEY_SECRET=your-razorpay-secret

# CORS
CLIENT_URL=http://localhost:5173
```

### Frontend Configuration

Create a `.env` file in `client/user/`, `client/organizer/`, and `client/admin/`:

```env
VITE_API_URL=http://localhost:5000/api
VITE_FIREBASE_API_KEY=your-firebase-key
VITE_FIREBASE_AUTH_DOMAIN=your-auth-domain
VITE_FIREBASE_PROJECT_ID=your-project-id
```

## рџ“љ API Documentation

### Base URL
```
http://localhost:5000/api
```

### Authentication Endpoints

#### Register User
```http
POST /user/auth/register
Content-Type: application/json

{
  "name": "John Doe",
  "email": "john@example.com",
  "password": "securePassword123"
}
```

#### Login User
```http
POST /user/auth/login
Content-Type: application/json

{
  "email": "john@example.com",
  "password": "securePassword123"
}
```

#### Organizer Register
```http
POST /organiser/auth/register
Content-Type: application/json

{
  "name": "Event Hub Co",
  "email": "organizer@example.com",
  "password": "securePassword123"
}
```

### Event Endpoints

#### Create Event (Organizer)
```http
POST /organiser/events/create
Authorization: Bearer <token>
Content-Type: multipart/form-data

{
  "title": "Tech Summit 2024",
  "description": "Annual tech conference",
  "eventType": "expert-talk",
  "date": "2024-04-15",
  "time": "10:00",
  "location": "San Francisco, CA",
  "capacity": 500,
  "banner": <file>
}
```

#### Get All Events
```http
GET /user/events
```

#### Get Event Details
```http
GET /user/events/:eventId
```

#### Update Event (Organizer)
```http
PUT /organiser/events/:eventId
Authorization: Bearer <token>
```

#### Delete Event (Organizer)
```http
DELETE /organiser/events/:eventId
Authorization: Bearer <token>
```

### User Endpoints

#### Get User Profile
```http
GET /user/profile
Authorization: Bearer <token>
```

#### Update User Profile
```http
PUT /user/profile
Authorization: Bearer <token>
Content-Type: application/json
```

#### Get User Events
```http
GET /user/my-events
Authorization: Bearer <token>
```
